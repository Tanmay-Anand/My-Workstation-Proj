# My Workstation

My Workstation is a full-stack personal productivity hub. It lets you manage notes, bookmarks, links, and tasks in one place.  
The project is built as a monorepo and contains both backend (Spring Boot) and frontend (React + Vite) codebases.

This repository is structured for easy development, clean deployment, and clear separation of concerns.

---

Project Structure:
my-workstation/
├─ backend/ # Spring Boot (Web, JPA, Validation, Security, PostgreSQL)
└─ frontend/ # React + Vite + TailwindCSS + Axios + Router


---

Technologies Used

### Backend
- Java 17+
- Spring Boot
- Spring Web
- Spring Data JPA
- Spring Security (later phases)
- PostgreSQL
- Lombok

### Frontend
- React 18 (Vite)
- React Router
- TailwindCSS
- Axios
